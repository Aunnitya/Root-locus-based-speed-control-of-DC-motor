# Root-locus-based-speed-control-of-DC-motor
## Project Overview
This project focuses on speed control of a DC motor using Root Locus techniques and MATLAB/Simulink.

## Objective
- Model a DC motor mathematically
- Design controller using Root Locus
- Implement P and PI controllers
- Analyze settling time, overshoot and steady-state error

## Tools Used
- MATLAB
- Simulink
- Control Systems concepts

## Team Contribution
- Simulink model development
- Mathematical modeling
- Control analysis and simulation results
